# World Theology University Website

## Características principales

- **Diseño profesional y moderno**: Inspirado en pillar.edu, pero con identidad visual y contenido 100% original.
- **Responsive**: Adaptado para móviles, tablets y escritorio usando Bootstrap 5 local.
- **Navbar dinámico**: Menú superior fijo, con enlaces a todas las secciones y página interna de cursos/certificados.
- **Traducción dinámica**: Botón para alternar entre español e inglés en todo el sitio, incluyendo menús y tarjetas.
- **Animaciones suaves**: Efecto fade-in en secciones y tarjetas al hacer scroll.
- **Catálogo de cursos y diplomados**: Página interna con cards detalladas, traducibles, para cada curso y diplomado.
- **Footer con redes sociales**: Enlaces a WhatsApp, Facebook, X (Twitter) e Instagram.
- **Colores y branding personalizados**: Paleta azul, blanco y gris, con clases utilitarias para fondo y texto.
- **Separación de recursos**: CSS y JS personalizados en archivos separados (`static/css/style.css` y `static/js/script.js`).
- **Accesibilidad**: Uso de etiquetas semánticas, contraste adecuado y navegación clara.
- **Integración de FontAwesome**: Íconos modernos para menús, cards y redes sociales.
- **Fácil mantenimiento**: Estructura clara de carpetas y archivos, código comentado y reutilizable.

## Estructura de carpetas

- `index.html` — Página principal
- `courses-certificates.html` — Catálogo de cursos y diplomados
- `static/css/style.css` — Estilos personalizados
- `static/js/script.js` — Lógica de animación y traducción
- `static/img/` — Imágenes y logotipo
- `static/bootstrap/` — Bootstrap 5 local y ejemplos

## Funcionalidades destacadas

- Cambio de idioma instantáneo sin recargar la página
- Cards de cursos y diplomados con información relevante y botón de inscripción vía WhatsApp
- Footer con links rápidos y redes sociales
- Código preparado para ampliación de secciones y nuevos idiomas
